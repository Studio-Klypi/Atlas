# Atlas

Backoffice unifié de **Studio Klypi** : CRM, facturation, projets, contenus (blog/FAQ), newsletter, tickets et rendez-vous.

## ⚙️ Stack
- Nuxt 4, Tailwind CSS v4, shadcn-vue
- PostgreSQL + Prisma

## 🚀 Démarrage
```bash
yarn install
cp .env.example .env
# Configure DATABASE_URL
yarn db:setup
```
